# Dart Deep Foundation

## Purpose
This repository is created to intentionally revise and strengthen Dart fundamentals in order to build more stable, scalable, and maintainable Flutter applications.  
The current focus is on understanding core Dart concepts and updates in the latest Dart versions.

## What This Repository Contains
This repository is organized into the following folders:

- `docs` – Learning notes, explanations, and roadmaps
- `src` – Dart source files for practicing language fundamentals
- `test` – Assignments and small tasks to validate understanding
- `assets` – Reference materials and supporting documents related to Dart concepts

## Learning Approach
- Daily focused learning of Dart fundamentals
- Hands-on practice by writing small programs
- Reinforcement through problem-solving and edge-case scenarios to improve clarity and confidence

## How to Run Code
Each Dart file can be executed using the Dart CLI.

Example command:
```bash
dart run file_name.dart
